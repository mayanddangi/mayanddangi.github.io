About

Hello !

I’m a graduate student at the University of Wisconsin–Madison, working in the field of quantum sensing. I hold a B.Tech in Engineering Physics from IIT Delhi and have hands-on experience with MOCVD of III-Nitrides, embedded systems, lasers, optics, and electronics. I’m deeply interested in quantum technologies and semiconductors. I also enjoy exploring the intersection of software development and AI with experimental research.

Beyond academics, I’m an avid sports enthusiast and have actively participated in a variety of sports. In my free time, I love to express my creativity through painting and poetry.

Projects

Designing of optical tweezers

This project focuses on building of optical tweezers to capture an array of single neutral atoms crucial for quantum computing and to develop a software routine to sort out atom for defect free array of atoms. The project github link can be found here.

AXLR8R Formula Racing

We had designed custom embedded hardware based on STM32F0 and STM32F4 microcontroller for intra-vehicle comm. and for managing & manipulating the received data from various installed sensors and related software.

External Cavity Diode Lasers

We have built our own external cavity diode lasers with custom designed mechanical housing for 852nm.

Laser Locking System

This project aims for the development of a low cost laser frequency locking based on STM32 microcontroller for ECDLasers to be used in cold atoms labs.

Chandrayaan-2 SAR image orthorectification

I have worked on removing geometrical distortions of SAR images caused by its side looking property using backward range doppler geocoding.

Processor from scratch using Verilog

In this project I did building a processor over FPGA by writing custom instruction set architecture (ISA), including arithmetic, logic, and conditional instructions.

Syncronous circuit simulation using Verilog

As a part of coursework, I built Designed and code synchronous circuit such as 4-Bit Gray code counter, Ring Counter and FSM in Verilog using S-R and D fip-fops & simulated the Verilog code using GTKWave.

Image classifier

Image classification implemented using convolution neural network using PyTorch

Image classification using decision tree

Scratch implementation of the decision tree model for binary classifcation.

Numerical solutions of the time-dependent Schrödinger equation

This contains the code for obtaining accurate numerical solutions of the time-dependent Schrödinger equation based on this paper in Julia.

Computational Optical Imaging

This contains the script for field propagation using angular spectrum, weiner filteration, Image reconstruction and deblurring using optimization technique and TV denoising.
